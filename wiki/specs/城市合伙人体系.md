## 五、怎么算钱：两段计费详解

### 第一阶段：预购（你的余额还有钱）

每当一名学员报名，总部从你的预购余额中扣除对应的拿货成本，差价全部归你。

**举例（银牌合伙人，7 折）：**

```
招收一名青三阶学员：
  零售价 ¥39,800
  拿货成本 ¥27,860（7折）
  余额扣除 ¥27,860
  你赚     ¥11,940
```

### 边界情况：余额不够扣一整门课怎么办？

当剩余余额不足以支付当前学员对应的课程成本时，该名学员**整单进入分账模式**，不拆分余额。

> 例：银牌余额还剩 ¥2,000，但下一名学员报青三阶（成本 ¥27,860）→ ¥2,000 不够扣 → 这单不进余额，直接走超额分账（你拿 20%）。

### 第二阶段：超额分账（余额用完后）

余额用完后，每招收一名学员，学费按比例直接分成。

**举例（银牌合伙人，你:总部 = 2:8）：**

```
招收一名青三阶学员：
  零售价 ¥39,800
  你拿 20% = ¥7,960
  总部拿 80%
```

### 同一门课，两段收益差距

| 课程 | 预购阶段你赚（7折） | 分账阶段你赚（20%） | 差距 |
|------|:--:|:--:|:--:|
| 青一阶 ¥6,800 | ¥2,040 | ¥1,360 | **¥680** |
| 青三阶 ¥39,800 | ¥11,940 | ¥7,960 | **¥3,980** |

> 预购永远比超额多得 10 个百分点。这就是为什么你应该买更大的包。

## 六、累积升级规则

### 三条规则，极其简单

**规则一：买了就是你的，档位只升不降。**

升级后折扣、分账比例永久保留。没有考核、没有降级、没有维持线。

**规则二：12 个月内累积升级。**

从签约日起 12 个月内，任何新增采购都计入累计。触发下一档门槛 → 即时升级。

```
银牌(¥10万)
  │ 再购 ¥10万，累计 ¥20万 → 升级金牌
  │
金牌(¥20万)
  │ 再购 ¥10万，累计 ¥30万 → 升级钻石
  │
钻石(¥30万)
  │ 再购 ¥20万，累计 ¥50万 → 升级战略
  │
战略(¥50万) → 已封顶
```

升级当天即刻生效三件事：折扣变深、分账比例提高、手中剩余余额等额升值（同样余额能覆盖更多课程）。

**规则三：每个周年日累计清零，档位不变。**

签约周年日（如 3月15日签约 → 次年 3月15日），12 个月内的新增采购累计数归零，但你的档位不动。新周期想继续升级？冲下一档门槛即可。

| 你现在是 | 想升下一档 | 新周期需要累计新增采购 |
|:--:|:--:|:--:|
| 银牌 | 金牌 | ¥10 万 |
| 金牌 | 钻石 | ¥10 万 |
| 钻石 | 战略 | ¥20 万 |

### 余额有效期

预购余额自购买之日起 **24 个月有效**。24 个月内消耗超过 30% → 自动顺延 12 个月。持续在招生、持续在消耗，余额就持续有效。

## 七、补贴与扶持

以下所有补贴均以「学员已完成课程交付」为结算前提。仅报名未上课不计入。

### 7.1 一阶引流专项补贴

一阶是漏斗入口。推广一阶课程，为二阶三阶输送生源，总部按每名完成交付的一阶学员额外补贴：

| 你的档位 | 每名一阶学员补贴 |
|:--:|:--:|
| 银牌 | ¥800 |
| 金牌 | ¥600 |
| 钻石 | ¥400 |
| 战略 | 无（5 折利润已足够） |

> 含青少年一阶和成人一阶。按季度结算（详见第十二章）。

### 7.2 首期开营扶持金

首次组织开营，总部承担 **50% 的场地费用**，上限 ¥5,000。仅首期有效，助你冷启动。

### 7.3 年度达标返利

按年度实际消耗的预购金额（即你从余额里真实扣掉的钱，反映真实业务量）计算：

| 年消耗预购金额 | 返利比例（基础包的 %） |
|:--:|:--:|
| ≥ ¥10 万 | 5% |
| ≥ ¥20 万 | 8% |
| ≥ ¥40 万 | 12% |

> 例：初始基础包 ¥20 万，年度实际消耗 ¥25 万（达 ≥¥20 万档）→ ¥20 万 × 8% = ¥16,000 返利。周年日结算，可直接抵扣次年采购或现金提取。

> 为什么按消耗金额而非学员数？学员买的课程不同（一阶 ¥6,800 vs 三阶 ¥39,800），按金额衡量更公平，反映真实业务量。

## 九、城市分级、保证金与独家保护

### 9.1 城市分级

城市等级为建议标准，最终档位由合伙人根据自身资源和意愿自选。选高不选低——折扣和分账比例跟着投入走，不跟着城市走。

| 城市等级 | 代表城市 | 建议基础包 |
|:--:|------|:--:|
| 一线 | 北京/上海/广州/深圳 | ¥30-50 万 |
| 新一线/省会 | 杭州/成都/武汉/重庆 等 | ¥20-30 万 |
| 二三线 | 珠海/无锡/青岛 等 | ¥10-20 万 |

### 9.2 保证金

所有合伙人统一缴纳 **¥3 万保证金**。合作终止时满足以下全部条件 → 全额退还：

1. 无恶意价格倾销（低于零售指导价 8 折销售）
2. 未将品牌及课程资料用于自有品牌或第三方
3. 未主动挖猎总部讲师及核心团队
4. 未以总部名义签署未经授权的商务合同

> 以上四条均为明确的「是/否」判断，不设模糊裁量空间。合作终止时 30 个工作日内完成保证金结算。

### 9.3 独家城市保护

| 条件 | 权益 |
|------|------|
| 基础包 **≥ ¥30 万**（钻石及以上） | 所在城市 **12 个月独家运营权** |
| 独家期内年消耗预购金额 **≥ ¥15 万** | 独家权自动续期 12 个月 |
| 独家期内年消耗预购金额 < ¥15 万 | 转为非独家（仍保留该城市运营权，可继续招生） |

年消耗金额以独家权生效日起 12 个月为周期计算。

> 非独家城市不限制合伙人数量，但同一渠道（如同一所学校）遵循「先报备先保护」原则。
