这个约束在任何档位、任何场景下都不会被打破。验证：铜牌进货利润率 30%，分润利润率 20%，差 10pp；战略进货利润率 50%，分润利润率 40%，同样差 10pp。全部成立。

## 五、怎么算钱：两段计费详解

### 第一阶段：配课（你的课程余额还有钱）

每当一名学员报名，总部从你的课程余额中扣除对应的配课成本，差价全部归你。

**举例（铜牌合伙人，7 折）：**

```
招收一名青三阶学员：
  总部指导价 ¥39,800
  配课成本 ¥27,860（7折）
  余额扣除 ¥27,860
  你赚     ¥11,940
```

### 边界情况：余额不够扣一整门课怎么办？

当剩余余额不足以支付当前学员对应的课程成本时，该名学员**整单进入分润模式**，不拆分余额。

> 例：铜牌余额还剩 ¥2,000，但下一名学员报青三阶（成本 ¥27,860）→ ¥2,000 不够扣 → 这单不进余额，直接走超额分润（你拿 20%）。

### 第二阶段：超额分润（余额用完后）

余额用完后，每招收一名学员，学费按比例直接分成。

**举例（铜牌合伙人，你:总部 = 2:8）：**

```
招收一名青三阶学员：
  总部指导价 ¥39,800
  你拿 20% = ¥7,960
  总部拿 80%
```

### 同一门课，两段收益差距

| 课程（总部指导价） | 配课阶段你赚（7折） | 分润阶段你赚（20%） | 差距 |
|------|:--:|:--:|:--:|
| 青一阶 ¥8,800 | ¥2,640 | ¥1,760 | **¥880** |
| 青三阶 ¥39,800 | ¥11,940 | ¥7,960 | **¥3,980** |

> 配课永远比分润多 10 个百分点。这就是为什么你应该买更大的包。

## 六、累积升级规则

### 三条规则，极其简单

**规则一：买了就是你的，档位只升不降。**

升级后折扣、分润比例永久保留。没有考核、没有降级、没有维持线。

**规则二：12 个月内累积升级。**

从签约日起 12 个月内，任何新增采购都计入累计。触发下一档门槛 → 即时升级。

```
铜牌(¥10万)
  │ 再购 ¥10万，累计 ¥20万 → 升级银牌
  │
银牌(¥20万)
  │ 再购 ¥10万，累计 ¥30万 → 升级金牌
  │
金牌(¥30万)
  │ 再购 ¥10万，累计 ¥40万 → 升级钻石
  │
钻石(¥40万)
  │ 再购 ¥10万，累计 ¥50万 → 升级战略
  │
战略(¥50万) → 已封顶
```

升级当天即刻生效三件事：配课折扣降低、分润比例提高、手中剩余余额等额升值（同样余额能覆盖更多课程）。

**规则三：每个周年日累计清零，档位不变。**

签约周年日（如 3月15日签约 → 次年 3月15日），12 个月内的新增采购累计数归零，但你的档位不动。新周期想继续升级？冲下一档门槛即可。

| 你现在是 | 想升下一档 | 新周期需要累计新增采购 |
|:--:|:--:|:--:|
| 铜牌 | 银牌 | ¥10 万 |
| 银牌 | 金牌 | ¥10 万 |
| 金牌 | 钻石 | ¥10 万 |
| 钻石 | 战略 | ¥10 万 |

### 余额有效期

课程余额自购买之日起 **12 个月有效**。12 个月内消耗超过 30% → 自动顺延 12 个月。持续在招生、持续在消耗，余额就持续有效。

## 九、城市分级、保证金与独家保护

### 9.1 城市分级

城市等级为建议标准，最终档位由合伙人根据自身资源和意愿自选。选高不选低——折扣和分润比例跟着投入走，不跟着城市走。

| 二三线 | 珠海/无锡/青岛 等 | ¥10-20 万 |

### 9.2 保证金

所有合伙人统一缴纳 **¥3 万保证金**。合作终止时满足以下全部条件 → 全额退还：

1. 无恶意价格倾销（低于总部指导价销售）
2. 未将品牌及课程资料用于自有品牌或第三方
3. 未主动挖猎总部讲师及核心团队
4. 未以总部名义签署未经授权的商务合同

> 以上四条均为明确的「是/否」判断，不设模糊裁量空间。合作终止时 30 个工作日内完成保证金结算。

### 9.3 战略合伙人网络引流

每个城市最多 **3 名** 战略合伙人。战略合伙人享受总部网络引流权益：

总部通过官网、社交媒体、线上推广等渠道持续招生引流。当总部收到你所在城市的学员咨询或报名时：
- 首次课程学费全额归属总部，你协助本地对接
- 首次课程完成后，该学员的客户关系移交给你——后续续报、升阶、转介绍产生的全部收益按你的档位正常结算
- 当城市内有2-3名战略合伙人时，网络线索按轮转原则公平分配

保持权益的条件：每月招收至少 **3 名**学员。单月不足3名 → 引流权益暂停，恢复招生后自动恢复。

铜牌至钻石档位共享城市运营权，同一渠道遵循「先报备、先保护」原则。
