## 二、4种决策时间节点 × 实际影响

### 场景A：7/7前决策（最佳）

| 阶段 | 时间 | 做什么 |
|------|------|--------|
| 7/7-7/13 | 1周 | 工具账号注册+讲师确认+物料制作启动 |
| 7/14-7/27 | 2周 | 合伙人签约+招生物料到位+试讲排练 |
| 7/28-8/17 | 3周 | 招生冲刺期（目标30-40人） |
| 8/18-8/31 | 2周 | 招生收尾+分组+问卷+最终准备 |
| 9/1-9/28 | 4周 | 试讲+助教培训+场地布置+彩排 |
| 10/1 | - | 开营 |

**总缓冲：充裕，每个环节有余量**

### 场景B：7/14前决策（可接受）

| 影响项 | 损失 |
|--------|------|
| 合伙人签约期 | 从2周压缩到1周（部分合伙人可能赶不上首批） |
| 招生启动 | 晚1周，冲刺期从3周变2周 |
| 预估首期人数 | 从30-40人降至20-30人 |
| 收入影响 | 基准场景¥16.2万 → 可能降至¥12-14万 |
| 物料制作 | 需要加急（可能产生外包加急费¥500-1000） |

**结论：可行，但招生目标需下调**

### 场景C：7/21前决策（危险）

| 影响项 | 损失 |
|--------|------|
| 合伙人签约 | 首批4城可能只有2城来得及 |
| 招生冲刺期 | 压缩到仅1.5周 |
| 预估首期人数 | 15-20人（仅能维持底线场景） |
| 试讲排练 | 从4周压缩到2周，质量风险↑ |
| 讲师备课 | 勉强够，但没有纠错时间 |
| 收入影响 | 底线场景¥10.8万，勉强覆盖成本 |

**结论：勉强能开，但质量/规模/利润三缩水**

### 场景D：7/28后决策（红线后）

| 影响项 | 后果 |
|--------|------|
| 招生窗口 | 不足5周，难以完成30人目标 |
| 合伙人渠道 | 首批大概率来不及 |
| 试讲质量 | 讲师仅2-3周备课，教案消化不充分 |
| 物料品质 | 赶工出的物料影响品牌形象 |
| 建议 | **推迟至10月下旬或元旦档期** |

**结论：10/1开营大概率延期**

## 四、各阻塞项的独立影响

### D1 工具账号（影响度：⭐⭐⭐）

- **现状：** 操作手册已就绪，11个免费工具学员自注册
- **不注册的后果：** 开营当天现场注册 → 浪费30-60分钟教学时间 → 教学进度被打乱
- **最晚启动日：** 8/25（开营前5周发注册指南）
- **今天启动的好处：** 有充足时间测试每个工具、排查问题、准备备用方案

### D2 讲师团队（影响度：⭐⭐⭐⭐⭐）

- **现状：** 未确认任何人选
- **不确认的后果：** 
  - 无法启动备课 → 试讲排练无法进行
  - 教案2500+行需要2-3周消化 → 7/11前确认才能保证9月有4周排练
  - 助教无法培训（助教培训依赖讲师主导）
- **最晚确认日：** 7/11（否则排练期不足4周）
- **⚠️ 这是唯一有硬性deadline的决策项**

### D3 招生目标（影响度：⭐⭐⭐）

- **现状：** 方案建议40人（乐观）/30人（基准）/20人（底线）
- **不确认的后果：** 渠道策略无法启动（不知道要招多少人→不知道每个渠道分配多少）
- **最晚确认日：** 7/18（招生启动前1周）
- **建议：** 如果纠结，先按30人启动，后面可以调
