## 一、首期招生总目标

### 1.1 核心目标

| 维度 | 目标值 | 底线值 | 说明 |
|------|--------|--------|------|
| 首期总学员 | **40人** | **20人** | 20人是开营最低门槛 |
| 青少年赛道 | 20人 | 10人 | 单独成班 |
| 成人赛道 | 20人 | 10人 | 单独成班 |
| 开课阶数 | **仅一阶** | 仅一阶 | 首期聚焦一阶验证，不同时开二三阶 |

### 1.2 为什么首期只开一阶

| 理由 | 说明 |
|------|------|
| 验证模式 | 一阶是引流款，先验证招生→交付→口碑的完整闭环 |
| 降低风险 | 一阶青少年5天/成人3天，场地/讲师/物料成本最低，试错成本可控 |
| 建立口碑 | 一阶学员是二三阶的种子用户，先做好一阶再升阶 |
| 团队磨合 | 首期交付让讲师/助教/运营团队跑通协作流程 |
| 快速迭代 | 1天课程便于快速复盘改进，7天就能出下一期 |

### 2.2 渠道详细策略

#### 渠道A：城市合伙人渠道（目标40%，12人）

**为什么是第一渠道：**
- 合伙人有本地学校/机构/商学院资源
- 合伙人自带信任背书（家长/职场人更信熟人推荐）
- 验证合伙人模式可行性（投资人关心的核心问题）

**执行策略：**

| 步骤 | 时间 | 行动 | 产出 |
|------|------|------|------|
| 1 | 7/14-7/20 | 确定首批2-3个试点城市合伙人 | 签约名单 |
| 2 | 7/21-7/27 | 合伙人赋能培训（招生话术/物料使用） | 培训完成 |
| 3 | 7/28-8/3 | 合伙人启动本地招生 | 各城3-5人目标 |
| 4 | 8/4-8/31 | 每周同步招生进度，提供支援 | 每周数据更新 |

**各城分配目标（基准30人）：**

| 城市 | 合伙人类型 | 青少年 | 成人 | 合计 | 说明 |
|------|-----------|--------|------|------|------|
| 海南 | 运营合伙人 | 3人 | 2人 | 5人 | 总部所在地，直营为主 |
| 山东 | 资源合伙人 | 2人 | 1人 | 3人 | 学校资源丰富 |
| 重庆 | 资源合伙人 | 1人 | 1人 | 2人 | 职场人群密集 |
| 珠海 | 资源合伙人 | 1人 | 1人 | 2人 | 珠三角辐射 |
| **合计** | | **7人** | **5人** | **12人** | |

> **注意：** 首期合伙人的核心任务不是招满人，而是**验证渠道可行性**。哪怕只招到2-3人，只要转化率和满意度达标，就说明合伙人模式跑通了。

## 四、招生转化策略

### 4.1 转化漏斗设计

```
朋友圈/公众号/合伙人曝光
  │
  ├─ 咨询入口（微信/电话/公众号留言）
  │    │
  │    ├─ 首次接触（30秒判断需求→分流）
  │    │    │
  │    │    ├─ 青少年家长 → 卖未来竞争力
  │    │    ├─ 职场人 → 卖真实生产力
  │    │    └─ 企业老板 → 卖团队效率
  │    │
  │    ├─ 需求摸底（三问：做什么工作/想解决什么/之前学过AI吗）
  │    │
  │    ├─ 课程推荐（根据需求推荐最合适的阶段）
  │    │
  │    ├─ 异议处理（10大高频异议，见《招生转化话术SOP.md》）
  │    │
  │    └─ 促成报名（5种促成法，见《招生转化话术SOP.md》）
  │
  └─ 缴费确认（24h/48h/72h跟进，见《招生转化话术SOP.md》）
```

### 4.2 转化率目标

| 漏斗层级 | 目标转化率 | 行业参考 | 说明 |
|---------|-----------|---------|------|
| 曝光→咨询 | 2-5% | 1-3% | 朋友圈/公众号/合伙人推广 |
| 咨询→有效咨询 | 70% | 60-80% | 排除同行/无效/试探 |
| 有效咨询→意向 | 50% | 40-60% | 需求匹配+课程推荐 |
| 意向→报名 | 60% | 50-70% | 异议处理+促成 |
| 报名→缴费 | 85% | 75-90% | 24h内跟进 |
| **全链路（咨询→缴费）** | **≥30%** | **15-25%** | 核心KPI |

### 4.3 首期招生话术重点

**青少年赛道（面向家长）：**
- 核心卖点：孩子越早接触AI，未来竞争力越强
- 痛点切入：别的孩子已经在用AI了，你家孩子还在刷题
- 成果展示：当天带走AI海报+30秒视频+提示词模板
- 价格锚定：¥8,800 = 一台手机的钱，但能给孩子打开一个新世界

**成人赛道（面向职场人）：**
- 核心卖点：1天学会用AI解决你工作中的真实问题
- 痛点切入：你花3小时做的PPT，AI 10分钟就能搞定
- 成果展示：当天带走可直接用的AI提示词模板和工作流
- 价格锚定：¥3,980 = 一顿商务饭局的钱，但能让你效率翻倍

**话术完整版：** 已在《招生转化话术SOP.md》中输出，覆盖6层漏斗话术+10大异议处理+5种促成法。

- 周度复盘模板（数据表+亮点/问题/调整计划）
- 异常预警机制（3级预警阈值+4类应急预案）

## 七、应急预案

### 7.1 招生不达标预案

| 触发条件 | 预警级别 | 应对方案 |
|---------|---------|---------|
| 8/18（W3）累计报名<10人 | 🔴 严重 | 启动备选渠道：社群裂变/企业团购/体验营加场 |
| 8/25（W4）累计报名<15人 | 🔴 严重 | 老板决策：降低开营门槛/延期/合并赛道 |
| 8/31累计报名<20人 | 🔴 阻塞 | 不开营，延期至下一期，已报名学员保留+补偿 |

### 7.2 赛道不均衡预案

| 触发条件 | 应对方案 |
|---------|---------|
| 青少年远多于成人（>3:1） | 加大成人赛道推广力度，调整朋友圈内容配比 |
| 成人远多于青少年（>3:1） | 加大青少年赛道推广，重点推家长群/学校渠道 |
| 单赛道<5人 | 考虑合并赛道（统一教学，分组实操） |

### 7.3 合伙人渠道失效预案

| 触发条件 | 应对方案 |
|---------|---------|
| 8/11前无合伙人签约 | 降低合伙人门槛，先签约再赋能 |
| 8/18合伙人渠道贡献<3人 | 总部直接补位，加大朋友圈+私域力度 |
| 合伙人完全无产出 | 标记为"资源型合伙人"，下期再验证运营能力 |
