## 三、招生运营风险（🟡 重要）

> 阶段：8月招生期 | 负责人：运营团队+城市合伙人

### M1. 招生人数不足

| 项目 | 内容 |
|------|------|
| **风险等级** | 🔴 严重 |
| **触发条件** | 8/31 确认人数不足 15 人 |
| **影响范围** | 无法开营或开营体验差 |
| **应急预案** | 方案A：降低开营门槛（10人可开）；方案B：合并双赛道（青少年+成人混班）；方案C：延期至10月中旬 |
| **预警机制** | 招生看板设置三级预警：绿色(≥20人)、黄色(15-19人)、红色(<15人) |
| **关联文档** | `docs/招生数据看板模板.md` · `docs/招生转化话术SOP.md` |

### M2. 招生渠道失效

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |
| **触发条件** | 某渠道连续2周零转化 |
| **影响范围** | 招生进度落后 |
| **应急预案** | 9类渠道备选（朋友圈/微信群/合作机构/地推/短视频/小红书/转介绍/企业合作/体验营），单渠道失效立即切换 |
| **预警机制** | 周度复盘时检查各渠道转化率，低于1%的渠道标记预警 |
| **关联文档** | `docs/朋友圈日历排期表.md` |

### M3. 咨询转化率低

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |
| **触发条件** | 咨询→报名转化率低于10% |
| **影响范围** | 流量浪费，招生效率低 |
| **应急预案** | 检查话术SOP执行情况 → 优化异议处理 → 增加体验营/说明会场次 → 调整朋友圈内容策略 |
| **关联文档** | `docs/招生转化话术SOP.md` |

### M4. 学员退费

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |
| **触发条件** | 开营前7天内退费申请超过10% |
| **影响范围** | 实际开营人数不确定 |
| **应急预案** | 开营前7天无条件退款政策已设定；超量退费时启动候补名单机制；开营前3天最终确认人数 |
| **关联文档** | `docs/招生转化话术SOP.md`（退费话术） |

### M5. 问卷回收率低

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 一般 |
| **触发条件** | 开营前3天问卷回收率低于60% |
| **影响范围** | 分组不准确，影响教学体验 |
| **应急预案** | 4级处理：首次提醒→二次提醒→电话回访→开营当天现场填写精简版 |
| **关联文档** | `docs/课前学员调研问卷.md` · `docs/学员分组管理模板.md` |

### M6. 早鸟价/优惠滥用

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟢 低 |
| **触发条件** | 大量学员要求额外折扣 |
| **影响范围** | 定价体系混乱 |
| **应急预案** | 统一口径"开营前7天无条件退款，锁定名额无风险"，不额外打折；特殊情况需老板审批 |
| **关联文档** | `docs/招生转化话术SOP.md`（异议处理话术） |

### M7. 负面口碑/同行调研

| 项目 | 内容 |
|------|------|

## 四、教学交付风险（🟠 重要）

> 阶段：9-10月交付期 | 负责人：讲师+助教团队

### T1. Wi-Fi 全断

| 项目 | 内容 |
|------|------|
| **风险等级** | 🔴 严重 |
| **触发信号** | 学员大面积举手，工具无法加载 |
| **处理方式** | 立即切换备用热点（提前准备3个4G/5G热点分散覆盖） |
| **负责人** | 主助教 |
| **时间限制** | 3分钟内恢复 |
| **预防措施** | 开营前一天测试场地网络；提前下载离线素材包；每个工具准备离线演示录屏 |

### T2. 工具崩溃（单个）

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |
| **触发信号** | 学员反馈某工具打不开 |
| **处理方式** | 切换备用账号 → 若仍不行则跳过该工具用替代方案 |
| **负责人** | 巡场助教 |
| **时间限制** | 2分钟内解决 |
| **预防措施** | 每个工具准备1-2个备用共享账号；提前测试所有账号可用性 |

### T3. 工具崩溃（全部/平台级）

| 项目 | 内容 |
|------|------|
| **风险等级** | 🔴 严重 |
| **触发信号** | 多人同时反馈工具不可用 |
| **处理方式** | 启用离线演示方案：讲师录屏演示 + 学员跟做截图 + 课后补交成果 |
| **负责人** | 讲师+主助教 |
| **时间限制** | 5分钟内切方案 |
| **预防措施** | 每个模块准备离线演示视频；核心操作提前截图做成PDF教程 |

### T4. 投影/音响故障

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |
| **触发信号** | 画面消失/模糊/无声 |
| **处理方式** | 切换备用笔记本 → 用大屏电视 → 学员围看讲师屏幕 |
| **负责人** | 主助教 |
| **时间限制** | 3分钟内恢复 |
| **预防措施** | 提前准备备用笔记本+HDMI/USB-C转换头；开营前1小时完整测试 |

### T5. 讲师设备故障

| 项目 | 内容 |
|------|------|
| **风险等级** | 🔴 严重 |
| **触发信号** | 讲师电脑死机/蓝屏 |
| **处理方式** | 助教立即递上备用电脑（提前装好教案+所有工具书签） |
| **负责人** | 主助教 |
| **时间限制** | 2分钟内切换 |
| **预防措施** | 备用笔记本与讲师电脑配置一致；教案+工具书签同步到云盘 |

### T6. 进度大面积落后

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |
| **触发信号** | 多组学员未完成，模块时间用完 |
| **处理方式** | 简化任务要求（降低成果标准），确保每人有基础产出；课后提供补做指引 |
| **负责人** | 讲师+主助教 |
| **时间限制** | 实时调整 |
| **预防措施** | 每个模块预留10%缓冲时间；设置"最低合格线"和"展示级"两级标准 |

### T7. 学员身体不适/突发状况

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |
| **触发信号** | 学员举手/面色异常/晕倒 |
| **处理方式** | 引导至休息区，提供水和常用药；严重则立即拨打120 |
| **负责人** | 签到/后勤 |
| **时间限制** | 立即处理 |
| **预防措施** | 物料包备创可贴/常用药；提前了解最近医院位置；青少年赛道需家长紧急联系人 |

### T8. 学员冲突/投诉

| **预防措施** | 开场强调"互助学习"氛围；分组时避免已知冲突搭配 |

### T9. 学员提前离开

| 项目 | 内容 |

| **触发信号** | 学员收拾东西 |

| **预防措施** | 开场强调完整体验的重要性；课程中设置"成果展示"环节增加粘性 |

### T10. 停电

| 项目 | 内容 |
|------|------|
| **风险等级** | 🟡 重要 |

## 五、赛事风险（🟠 重要）

> 阶段：赛事举办期间 | 负责人：赛事组委会

### C1. 参赛人数不足

| 项目 | 内容 |
|------|------|
| **可能性** | 中 |
| **影响** | 降低赛事氛围，评审缺乏样本 |
| **应对措施** | L1强制参与（结营必做）；L2设开放赛道吸引外部学员；提前宣传造势 |

### C2. 作品质量参差不齐

| 项目 | 内容 |
|------|------|
| **可能性** | 高 |
| **影响** | 评审困难，优秀作品被埋没 |
| **应对措施** | 设"最低合格线"，低于线的作品不参与评奖；评审标准提前公布 |

### C3. 抄袭/代做

| 项目 | 内容 |
|------|------|
| **可能性** | 低 |
| **影响** | 品牌风险，赛事公信力受损 |
| **应对措施** | 要求提交工具使用录屏；现场答辩验证真实性；设举报通道 |

### C4. 评审争议

| 项目 | 内容 |
|------|------|
| **可能性** | 中 |
| **影响** | 口碑风险，学员/家长不满 |
| **应对措施** | 多维评审（讲师+合伙人+大众投票）；设申诉机制；评审标准量化透明 |

### C5. 赞助拉不到

| 项目 | 内容 |
|------|------|
| **可能性** | 高 |
| **影响** | 赛事收入低于预期 |
| **应对措施** | 首期不依赖赞助收入；赛事成本控制在低成本范围（L1零成本，L2季度低成本） |

### C6. 城市合伙人组织能力不足

| 项目 | 内容 |
|------|------|
| **可能性** | 中 |
| **影响** | 赛事体验差，影响品牌 |
| **应对措施** | 总部提供全套物料+流程SOP；首期总部派人现场支持；赛事组织纳入合伙人考核 |
