## 八、异常预警机制

### 预警阈值

| 预警级别 | 触发条件 | 应对动作 |
|---------|---------|---------|
| 🟢 正常 | 周咨询量 ≥ 15，周报名 ≥ 4 | 按计划执行 |
| 🟡 关注 | 周咨询量 < 10 或 周报名 < 2 | 加大渠道投放，增加朋友圈频率，启动二次跟进 |
| 🔴 预警 | 连续2周报名 < 2 或 距开营2周仅完成 < 50% | 启动应急预案：降低门槛/增加体验营/合伙人加力/延期讨论 |

### 应急预案

| 场景 | 触发条件 | 方案 |
|------|---------|------|
| 青少年招生不足 | 开营前2周 < 10人 | ①合伙人加大力度 ②增加体验营转化 ③降低早鸟价门槛 |
| 成人招生不足 | 开营前2周 < 10人 | ①企业客户定向邀约 ②增加免费说明会场次 ③考虑合并赛道 |
| 合伙人渠道失效 | 单城招生 < 3人 | ①总部直补该城招生 ②调整合伙人策略 ③后续期次重新评估 |
| 总招生严重不足 | 开营前1周 < 20人 | ⏳ 老板决策：延期/缩小规模/转线上 |

## 九、使用说明

### 谁来填

| 角色 | 职责 | 频率 |
|------|------|------|
| 运营/客服 | 每日更新"每日更新记录"表 | 每天 |
| 唐僧 | 更新渠道来源、漏斗数据、周复盘 | 每周五 |
| 老板 | 查看核心指标一览、确认异常决策 | 按需 |

### 怎么填

1. **每日：** 有新咨询/报名/缴费时，更新"每日更新记录"表
2. **每周五：** 更新"渠道来源追踪""转化漏斗""学员画像"，填写"周度复盘模板"
3. **异常时：** 核对"异常预警机制"，触发预警时执行对应方案

### 数据来源

| 数据 | 来源 |
|------|------|
| 咨询量 | 微信聊天记录 / 报名表单后台 |
| 报名量 | 报名表单 / 小程序后台 |
| 缴费量 | 微信/支付宝收款记录 |
| 渠道来源 | 咨询时询问"从哪里看到的" |
| 学员画像 | 课前调研问卷 |

1. 复制本文件为 `招生数据看板-第X期.md`
2. 填写"核心指标一览"中的目标值
3. 招生启动后，每天更新"每日更新记录"
4. 每周五更新全部数据表

### 老板看什么

- **30秒版：** 只看"核心指标一览"（第一节）
- **5分钟版：** 加看"转化漏斗""周度复盘"
- **深挖版：** 看"渠道来源追踪""异常预警"
