## 一、品牌定位关键词

| 维度 | 关键词 |
|------|--------|
| 品牌调性 | 实战派、真实、专业、不装 |
| 情绪感受 | 有信心、有掌控感、有成果 |
| 视觉风格 | 科技感 + 实操感，不冷冰冰，有温度 |

**一句话视觉原则：** 让人看到就想动手，不是想听课。

## 四、Logo 使用规范

### 品牌名

- **主品牌名：** 未来
- **产品全称：** 未来 AI 实战训练营
- **英文参考：** Future AI Camp / EntroCamp（内部代号，对外不使用）

### Logo 构成建议

```
[图标] + 未来 AI 实战训练营
```

- **图标方向：** 抽象化的「启动/发射」符号（如火箭尾焰 + 光标箭头融合），或简洁的字母 F 变形
- **组合方式：** 横版（图标左 + 文字右）、竖版（图标上 + 文字下）
- **最小尺寸：** 横版宽度不小于 120px，竖版高度不小于 80px
- **安全区域：** Logo 四周留至少图标高度 1/4 的空白

### 禁止用法

- ❌ 不得拉伸变形
- ❌ 不得更改品牌色
- ❌ 不得加阴影、描边、渐变
- ❌ 不得在复杂背景上直接使用（需加底色块或蒙版）
- ❌ 不得将「未来」二字替换为其他字体风格

## 五、视觉元素库

### 5.1 图形语言

| 元素 | 说明 | 用途 |
|------|------|------|
| **终端窗口** | 模拟代码/命令行窗口样式 | 工具展示、实操场景 |
| **网格底纹** | 浅色细网格（2px线，#E2E8F0） | 背景装饰、科技感 |
| **光标/箭头** | 鼠标光标或代码光标闪烁符号 | 强调「动手操作」 |
| **进度条** | 三段式进度条（绿→蓝→紫） | 展示三阶成长路径 |
| **成果卡片** | 带边框的成品展示框 | 海报/短视频中展示学员成果 |
| **工具图标** | 各 AI 工具的简化图标 | 工具清单展示 |

### 5.2 禁用素材

- ❌ 通用机器人/AI概念图（太泛）
- ❌ 大脑神经网络抽象图（太虚）
- ❌ 纯蓝色科技渐变背景（太冷）
- ❌ 高度风格化的 3D 渲染（与实战调性不符）

### 5.3 推荐素材风格

- ✅ 真实操作界面截图（带终端/浏览器窗口）
- ✅ 学员实操现场照片（MacBook + 讲台）
- ✅ 成品作品展示（网站截图、海报成品、短视频画面）
- ✅ 扁平化插画（简洁线条 + 品牌色填充）

## 七、排版规范

### 7.1 海报排版结构

```
┌─────────────────────────┐
│  [品牌Logo]              │  ← 顶部左对齐
│                          │
│  ┌───────────────────┐  │
│  │   主标题（大字）    │  │  ← 视觉焦点
│  │   副标题（小字）    │  │
│  └───────────────────┘  │
│                          │
│  ┌───────────────────┐  │
│  │  成果展示区        │  │  ← 3个成果卡片
│  │  [卡片] [卡片] [卡片] │
│  └───────────────────┘  │
│                          │
│  ┌───────────────────┐  │
│  │  课程信息/日期/价格 │  │
│  └───────────────────┘  │
│                          │
│  ┌───────────────────┐  │
│  │  [CTA 按钮]        │  │  ← 醒目颜色
│  │  扫码报名/咨询     │  │
│  └───────────────────┘  │
│                          │
│  [底部：品牌信息/二维码] │
└─────────────────────────┘
```

### 7.2 社交媒体图排版

- **朋友圈九宫格：** 第1张为主视觉，第2-8张为单点卖点，第9张为二维码/联系方式
- **朋友圈单图：** 居中构图，上方品牌+主题，中间核心卖点（不超过3行），下方CTA

### 7.3 PPT 模板规范

- **封面页：** 品牌Logo + 课程名称 + 讲师信息
- **目录页：** 三色进度条（绿→蓝→紫）对应三阶内容
- **内容页：** 左侧文字（60%），右侧配图/截图（40%）
- **实操页：** 全屏截图/操作界面，底部半透明文字说明
- **成果页：** 居中展示成品截图 + 品牌色边框
- **结束页：** 品牌Logo + 下一步行动 + 联系方式

## 八、双赛道视觉区分

| 维度 | 青少年赛道 | 成人赛道 |
|------|-----------|----------|
| 主色调 | 品牌蓝 + 活力绿点缀 | 品牌蓝 + 科技蓝深色 |
| 图片风格 | 年轻、活力、创作场景 | 专业、高效、职场场景 |
| 字体感受 | 圆润、友好 | 稳重、专业 |
| 文案语气 | 「你的第一个AI作品」 | 「你的AI工作效率翻倍」 |
| CTA 文案 | 「开始创造」 | 「立即提升」 |

**注意：** 品牌主色不变，双赛道仅在辅助色、图片风格和文案语气上做区分。
