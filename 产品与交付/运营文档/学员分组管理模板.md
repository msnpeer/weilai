> 未来AI实战训练营 · 开营前核心工具
> 使用时机：问卷回收完成后、开营前5天
> 配套文档：《课前学员调研问卷》《讲师助教快速上手手册》《招生数据看板模板》

| 项目 | 说明 |
|------|------|
| 适用阶段 | 问卷回收截止后 → 开营前完成分组 |
| 操作人 | 城市运营负责人 / 班主任 |

## 一、分组决策树

```
收到问卷数据
    │
    ├── Step 1: 分赛道
    │   ├── 青少年赛道 → 进入青少年分组流程
    │   └── 成人赛道 → 进入成人分组流程
    │
    ├── Step 2: 分课程阶段
    │   ├── 一阶（破界营）→ 1天，无需细分小组
    │   ├── 二阶（造物营）→ 2天，按项目方向分组
    │   └── 三阶（掌舵营）→ 3天，按项目方向+技能水平分组
    │
    ├── Step 3: 确定小组规模
    │   ├── 一阶：整班教学，不分组（助教巡场即可）
    │   ├── 二阶：每组4-5人，配1名助教覆盖2-3组
    │   └── 三阶：每组3-4人，配1名助教覆盖2组
    │
    ├── Step 4: 组内搭配原则
    │   ├── 技能均衡：每组至少1名有基础的 + 1-2名零基础
    │   ├── 年龄均衡（青少年）：同组年龄差不超过3岁
    │   ├── 背景多元（成人）：尽量不同行业搭配
    │   └── 性别均衡：每组男女比例尽量接近
    │
    └── Step 5: 输出分组表 + 通知学员
```

## 四、一阶破界营分组说明

> 一阶为1天入门营，整班教学，不需细分项目小组

### 4.1 座位安排建议

| 布局 | 适用人数 | 说明 |
|------|---------|------|
| U型排列 | ≤ 15人 | 讲师在开口处，助教在后方巡场 |
| 分岛排列 | 16-30人 | 每4-5人一个岛（圆桌），助教每2个岛配1人 |
| 阶梯教室 | 30+人 | 前排放零基础学员（方便辅导），后排放有基础的 |

### 4.2 座位分配原则

| 原则 | 说明 |
|------|------|
| 零基础靠前 | 助教容易关注到 |
| 有基础穿插 | 每个区域至少1名有基础的，可以帮带 |
| 同伴效应 | 认识的朋友可以坐一起（降低焦虑），但不能扎堆 |
| 设备检查 | 自带设备的坐外侧（方便充电），借用设备的坐内侧 |

## 五、分组操作SOP

### 5.1 时间线

| 时间节点 | 动作 | 负责人 | 产出 |
|---------|------|--------|------|
| 开营前7天 | 发送问卷 | 运营 | 问卷链接 |
| 开营前5天 18:00 | 问卷回收截止 | 运营 | 原始数据 |
| 开营前5天 20:00 | 数据录入分组表 | 运营 | 学员信息汇总表 |
| 开营前5天 22:00 | 初步分组方案 | 运营+讲师 | 分组方案表v1 |
| 开营前4天 | 讲师审核分组 | 讲师 | 分组方案表v2 |
| 开营前3天 | 确认终版分组 | 运营负责人 | 分组方案表终版 |
| 开营前3天 | 助教领取分组名单 | 助教 | 各助教分组表 |
| 开营前2天 | 通知学员分组 | 运营 | 微信通知 |
| 开营当天 | 现场核实出席 | 运营+助教 | 最终出席表 |

### 5.2 分组决策流程

```
Step 1: 数据清洗
├── 剔除无效问卷（未完成/明显乱填）
├── 标记未回收问卷（电话回访）
└── 生成学员信息汇总表

Step 2: 技能评级
├── 根据评级规则对每个学员打分
├── 标记 A/B/C/D 四级
└── 重点关注 D 级学员（需额外辅导）

Step 3: 赛道分流
├── 青少年赛道：按年龄段 + 技能水平分组
├── 成人赛道：按行业 + 技能水平分组
└── 跨赛道：不同赛道不混组

Step 4: 小组组建
├── 每组技能均衡（A/B/C/D 分散）
├── 年龄/行业多元化
├── 性别比例尽量均衡
└── 特殊需求学员安置

Step 5: 助教分配
├── 每2-3组配1名助教
├── 有D级学员的组优先配经验助教
├── 助教提前熟悉组员信息
└── 助教领取分组表 + 学员背景简报

Step 6: 通知与确认
├── 微信私信通知学员分组结果
├── 告知组名、组员（可选）、助教联系方式
├── 确认出席（回复"确认"）
└── 未回复的电话跟进
```

### 5.3 未回收问卷处理

| 情况 | 处理方式 | 时间 |
|------|---------|------|
| 问卷未回收 | 微信私信提醒 + 问卷链接 | 开营前5天 |
| 仍未回收 | 电话回访，口头完成核心问题 | 开营前4天 |
| 电话未接 | 短信留言 + 标记"待确认" | 开营前4天 |
| 开营前3天仍未联系 | 标记"缺席风险"，通知讲师 | 开营前3天 |

**电话回访核心问题（精简版）：**

```
青少年赛道（对家长说）：
1. 孩子平时用电脑多吗？（熟练/一般/很少）
2. 用过AI工具吗？比如ChatGPT、文心一言？（用过/听说过/没听过）
3. 孩子最想学什么？（画图/做视频/建网站/其他）
4. 需要我们提供电脑吗？（自带/需要借）

成人赛道：
1. 您平时工作中用电脑多吗？（很多/一般/很少）
2. 了解AI吗？用过哪些AI工具？（用过/听说过/没听过）
3. 您最想解决工作中的什么问题？
4. 需要我们提供电脑吗？（自带/需要借）
```

## 附录：分组常见问题FAQ

**Q1: 一阶真的不需要分组吗？**
> 一阶是1天入门体验，整班教学+助教巡场就够。分组会增加复杂度，反而影响体验。如果人数超过30人，可以考虑分两个班，但班内仍不分小组。

**Q2: 如果某个技能等级的人特别多/特别少怎么办？**
> 如果A级（有基础）特别多：每组多放1个A，让他们当"小组导师"。
> 如果A级特别少：确保每组至少1个B级，助教重点覆盖无A/B的组。
> 如果D级（零基础）特别多：考虑开"基础加强班"，前30分钟单独辅导。

**Q3: 成人赛道要不要按行业分组？**
> 不要。同行业扎堆容易"聊业务不聊AI"。跨行业搭配反而能碰撞出更多应用场景。但可以在休息时间安排"行业交流环节"。

**Q4: 学员自带设备和借用设备的要不要分开坐？**
> 建议不分开。借用设备的学员坐在内侧（方便统一管理），自带的坐外侧（方便充电）。但不要刻意区分，避免借用设备的学员感到"特殊"。

**Q5: 如果学员强烈要求和朋友一组怎么办？**
> 青少年赛道：可以满足，但要确保该组技能均衡。朋友同组可以降低焦虑感。
> 成人赛道：一般不鼓励，跨背景搭配更有价值。但如果双方都坚持，可以满足。

**Q6: 分组后发现某组整体水平偏弱怎么办？**
> 开营前发现：调整1-2名成员到其他组。
> 开营后发现：助教重点关照该组，讲师在巡场时多停留。
