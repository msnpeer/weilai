## 五、常见问题

### Q1：所有工具都免费吗？
大部分工具完全免费或有足够免费额度支撑课堂使用。Claude Code 和 Codex 有免费额度限制，讲师需准备付费账号作为备用。

### Q2：学员用手机还是电脑？
- **一阶**：手机 + 电脑均可（手机操作豆包/即梦/剪映更方便，电脑排版 Canva 更方便）
- **二阶/三阶**：**必须带电脑**（建站、Agent、知识库都需要电脑操作）

### Q3：学员注册遇到问题怎么办？
课前 3 天发注册指南 + 课前 1 天电话回访确认。开营当天签到区设"注册帮助台"，助教现场协助。

### Q4：工具突然用不了（限流/维护）怎么办？
每个工具准备 2-3 个备用账号。教案中已标注每个模块的替代方案（如豆包不可用时切 DeepSeek）。

### Q5：OpenClaw 的官网地址？
