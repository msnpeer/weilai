# 学员成果物模板库

> 未来 AI 实战训练营 · 三阶课程学员成果物创建指南
> 供课堂直接使用：学员打开就能照着做，讲师/助教打开就能指导

---

## 使用说明

### 文档定位
本模板库覆盖三阶课程共 **9 项标准成果物**（每阶 3 项），为每项成果物提供：
- **起步模板**：学员可以照着改，降低从零开始的难度
- **创建步骤**：分步操作指引，零基础也能跟
- **质量自检表**：学员做完后对照打勾，自查是否达标
- **常用提示词**：直接复制到 AI 工具中使用的示例 prompt
- **双赛道示例**：青少年 / 成人赛道各一个选题示范

### 成果物总览

| 阶段 | 课程 | 成果物 1 | 成果物 2 | 成果物 3 |
|------|------|---------|---------|---------|
| 一阶 | AI 破界营 | AI 原创海报 | 30 秒 AI 短视频 | 个人提示词模板 |
| 二阶 | AI 造物营 | 可访问网站 | AI 建站 SOP | 专属 Agent |
| 三阶 | AI 掌舵营 | 知识库体系 | 自动化脚本 | AI 系统方案 |

## 成果物 1：AI 原创海报

### 起步模板

学员无需从零开始，用以下提示词让 AI 生成初稿，再修改即可：

**通用海报提示词模板：**
```
帮我设计一张海报，主题是「{你的主题}」。

要求：
1. 海报尺寸：竖版（适合手机屏幕）
2. 主标题：大字醒目，放在上方 1/3
3. 副标题：一行小字说明
4. 正文区：3-5 个要点，用图标+短句排列
5. 底部：行动号召（扫码/报名/了解更多）
6. 配色：主色用{蓝色/绿色/紫色}，背景深色
7. 风格：科技感、简洁、现代

请直接输出海报的完整设计描述。
```

**青少年赛道示范：**
```
帮我设计一张海报，主题是「我的AI学习之旅」。

要求：
1. 海报尺寸：竖版（适合手机屏幕）
2. 主标题：「我的 AI 学习之旅」大字醒目
3. 副标题：「从零基础到AI小达人」
4. 正文区：展示我今天学到的 3 个技能（AI对话、AI画图、AI做视频）
5. 底部：「未来AI训练营 · 第一期学员」
6. 配色：活力蓝+科技绿，背景深空黑
7. 风格：酷炫、年轻、有未来感
```

**成人赛道示范：**
```
帮我设计一张海报，主题是「AI赋能工作效率」。

要求：
1. 海报尺寸：竖版（适合手机屏幕）
2. 主标题：「用AI，1小时干完1天的活」
3. 副标题：「未来AI训练营 · 职场效率革命」
4. 正文区：3个效率提升场景（周报自动化、PPT秒生成、数据一键分析）
5. 底部：扫码报名 / 联系方式
6. 配色：专业蓝+深灰，简洁商务
7. 风格：专业、简洁、有说服力
```

### 创建步骤

| 步骤 | 操作 | 工具 | 时间 |
|------|------|------|------|
| 1 | 打开 AI 图片生成工具（即梦/MidJourney/DALL-E） | 浏览器 | 1 分钟 |
| 2 | 复制上面的提示词模板，填入你的主题 | - | 3 分钟 |
| 3 | 生成 3-4 张初稿，选最好的一张 | AI 工具 | 5 分钟 |
| 4 | 如果需要微调，用图片编辑工具调整文字/布局 | Canva/美图 | 10 分钟 |
| 5 | 导出为 PNG/JPG，检查清晰度 | - | 2 分钟 |
| 6 | 对照质量自检表打勾 | 本模板 | 2 分钟 |

### 质量自检表

- [ ] 海报有明确的主题（一句话能说清楚）
- [ ] 主标题大字醒目，3米外能看清
- [ ] 配色不超过3种主色，视觉协调
- [ ] 文字没有错别字
- [ ] 图片清晰，不模糊不拉伸
- [ ] 整体布局有层次感（标题→内容→底部）
- [ ] 已导出为可分享的格式（PNG/JPG）

**自评等级：**
- ✅ 达标：以上全部打勾
- ⭐ 展示级：排版美观 + 配色协调 + 有创意亮点

## 成果物 2：30 秒 AI 短视频

### 起步模板

**视频脚本模板（通用）：**

```
【开场 3秒】
画面：你的主题文字/Logo 飞入
旁白/文字：「{你的主题}」

【主体 20秒】
镜头1（5秒）：展示第一个要点
  - 画面：AI 工具截图/动画
  - 文字：要点1关键词

镜头2（5秒）：展示第二个要点
  - 画面：操作过程/效果对比
  - 文字：要点2关键词

镜头3（5秒）：展示第三个要点
  - 画面：最终成果展示
  - 文字：要点3关键词

镜头4（5秒）：亮点/总结
  - 画面：成果全景
  - 文字：核心结论

【结尾 7秒】
画面：品牌/个人信息
文字：「未来AI训练营 · 第一期」
旁白：感谢语/行动号召
```

**青少年赛道示范：**
```
主题：「AI帮我做了个游戏海报」

【开场 3秒】
画面：游戏手柄图标 + 标题飞入
文字：「AI帮我做了个游戏海报！」

【主体 20秒】
镜头1：我告诉AI「帮我画一个赛博朋克风格的游戏海报」
镜头2：AI 生成了第一版，颜色太暗
镜头3：我调整提示词，加了「霓虹灯+亮色」，生成了完美版本
镜头4：最终海报展示，酷炫效果

【结尾 7秒】
文字：「用AI，人人都能当设计师」
「未来AI训练营 · 青少年赛道」
```

**成人赛道示范：**
```
主题：「AI帮我3分钟做完周报」

【开场 3秒】
画面：电脑屏幕 + 标题
文字：「3分钟搞定周报」

【主体 20秒】
镜头1：打开AI工具，输入本周工作内容关键词
镜头2：AI 自动生成结构化周报初稿
镜头3：我微调了2处细节，导出最终版
镜头4：对比：手动2小时 vs AI 3分钟

【结尾 7秒】
文字：「明天上班就能用」
「未来AI训练营 · 职场赛道」
```

### 创建步骤

| 步骤 | 操作 | 工具 | 时间 |
|------|------|------|------|
| 1 | 确定视频主题（你今天做了什么/学到了什么） | 脑暴 | 3 分钟 |
| 2 | 用上面的脚本模板写出你的 30 秒脚本 | 文档工具 | 5 分钟 |
| 3 | 用 AI 视频工具（可灵/Kling/剪映）生成素材 | AI 工具 | 10 分钟 |
| 4 | 用剪映拼接素材，加上文字和配乐 | 剪映 | 10 分钟 |
| 5 | 预览检查：时长是否 30 秒左右、画面是否连贯 | 剪映 | 2 分钟 |
| 6 | 导出 MP4，检查文件大小（建议 < 50MB） | - | 1 分钟 |
| 7 | 对照质量自检表打勾 | 本模板 | 2 分钟 |

### 质量自检表

- [ ] 视频时长在 25-35 秒之间
- [ ] 画面连贯，没有黑屏/卡顿
- [ ] 有字幕或关键文字说明
- [ ] 主题清晰（一句话能说清楚这个视频讲什么）
- [ ] 画面不模糊，分辨率可接受
- [ ] 有开头和结尾，结构完整
- [ ] 已导出为 MP4 格式

**自评等级：**
- ✅ 达标：以上全部打勾
- ⭐ 展示级：有配乐 + 转场流畅 + 内容有亮点

## 成果物 3：个人提示词模板

### 起步模板

**提示词模板框架：**
```
# 我的 AI 提示词模板集

## 模板1：{场景名称}
**用途：** {这个提示词用来做什么}
**适用工具：** ChatGPT / DeepSeek / 豆包 / 通义千问
**提示词：**
```
你是一个{角色}，帮我完成{任务}。

要求：
1. {要求1}
2. {要求2}
3. {要求3}

输出格式：{格式要求}
```
**使用示例：** 我输入了{xxx}，AI输出了{xxx}

---

## 模板2：{场景名称}
...（重复上面的结构）
```

**青少年赛道示范（3 条基础模板）：**

```
# 我的 AI 提示词模板集 · 学习助手版

## 模板1：错题分析
**用途：** 把错题丢给AI，让它帮我分析为什么错
**适用工具：** DeepSeek / 豆包
**提示词：**
```
你是我的学习助手。我给你一道错题，请帮我：
1. 分析我可能错在哪里
2. 给出正确的解题思路
3. 总结这类题目的解题技巧
4. 再出一道类似的题让我练习

我的错题是：{粘贴题目}
我的错误答案是：{你的答案}
正确答案是：{正确答案}

### 起步模板

```html

<html lang="zh-CN">
<head>
    <meta charset="UTF-8">

        * { margin: 0; padding: 0; box-sizing: border-box; }
        body { font-family: 'Microsoft YaHei', sans-serif; line-height: 1.6; color: #333; }

            background: linear-gradient(135deg, #2563EB, #7C3AED);
            color: white;
            text-align: center;

        }
```

### 常用提示词

**让 AI 帮你建站：**
```
帮我创建一个个人网站，用纯HTML+CSS实现，不需要框架。

网站主题：{你的主题}
风格要求：{现代简洁/活泼创意/专业商务}

页面结构：
1. Hero区：大标题+副标题+背景渐变
2. 关于我：个人介绍
3. 作品展示：3-4个作品卡片

技术要求：
- 响应式设计（手机/电脑都能看）
- 使用CSS Grid或Flexbox布局
- 配色方案：{主色}+{辅助色}
- 所有代码写在一个HTML文件中

请直接输出完整的HTML代码。
```

**让 AI 帮你改样式：**
```
我有一段HTML代码，帮我优化样式：

{粘贴你的代码}

优化要求：
1. 改配色为{你想要的颜色}
```

```python
    for filename in os.listdir(folder_path):
        if filename.endswith('.md'):
            with open(os.path.join(folder_path, filename), 'r', encoding='utf-8') as f:
                content = f.read()
                questions.append({
                    'file': filename,
                    'content': content,
                    'date': filename.split('-')[0] if '-' in filename else 'unknown'
                })
    return questions

def categorize_by_subject(questions):
    """按学科分类"""
    categories = {'数学': [], '英语': [], '物理': [], '其他': []}
    keywords = {
        '数学': ['方程', '函数', '几何', '代数', '计算'],
        '英语': ['语法', '单词', '阅读', '作文', '翻译'],
        '物理': ['力学', '电学', '光学', '运动', '能量']
    }
    for q in questions:
        matched = False
        for subject, kws in keywords.items():
            if any(kw in q['content'] for kw in kws):
                categories[subject].append(q)
                matched = True
                break
        if not matched:
            categories['其他'].append(q)
    return categories

def generate_summary(categories):
    """生成整理报告"""
    report = f"# 错题整理报告\n\n"
    report += f"生成时间：{datetime.now().strftime('%Y-%m-%d %H:%M')}\n\n"
    for subject, questions in categories.items():
        if questions:
            report += f"## {subject}（{len(questions)}题）\n\n"
            for q in questions:
                report += f"### {q['file']}\n{q['content']}\n\n---\n\n"
    return report

def main():
    print("📚 错题整理脚本启动...")
    questions = load_error_questions("./错题/")
    print(f"📥 找到 {len(questions)} 道错题")

    categories = categorize_by_subject(questions)
    for subject, qs in categories.items():
        if qs:
            print(f"  {subject}: {len(qs)}题")

    report = generate_summary(categories)
    with open("./错题整理报告.md", "w", encoding="utf-8") as f:
        f.write(report)
    print("✅ 整理完成！报告已保存到「错题整理报告.md」")

if __name__ == "__main__":
    main()
```

```python
"""
批量数据处理脚本
功能：读取Excel/CSV数据，自动分析并生成报告
"""

import csv
from datetime import datetime

def load_csv(file_path):
    """读取CSV文件"""
    data = []
    with open(file_path, 'r', encoding='utf-8') as f:
        reader = csv.DictReader(f)
        for row in reader:
            data.append(row)
    print(f"📥 读取了 {len(data)} 条数据")
    return data

def analyze_data(data):
    """基础数据分析"""
    if not data:
        return {}

    analysis = {
        'total_records': len(data),
        'columns': list(data[0].keys()),
    }

    # 尝试数值列统计
    for col in analysis['columns']:
        try:
            values = [float(row[col]) for row in data if row[col]]
            analysis[f'{col}_sum'] = sum(values)
            analysis[f'{col}_avg'] = sum(values) / len(values)
            analysis[f'{col}_max'] = max(values)
            analysis[f'{col}_min'] = min(values)
        except (ValueError, ZeroDivisionError):
            pass

    return analysis

def generate_report(data, analysis):
    """生成分析报告"""
    report = f"# 数据分析报告\n\n"
    report += f"生成时间：{datetime.now().strftime('%Y-%m-%d %H:%M')}\n\n"
    report += f"## 数据概览\n"
    report += f"- 总记录数：{analysis['total_records']}\n"
    report += f"- 字段数：{len(analysis['columns'])}\n"
    report += f"- 字段列表：{', '.join(analysis['columns'])}\n\n"

    report += f"## 数值统计\n"
    for key, value in analysis.items():
        if key.endswith('_avg'):
            col = key.replace('_avg', '')
            report += f"### {col}\n"
            report += f"- 总计：{analysis.get(f'{col}_sum', 'N/A'):.2f}\n"
            report += f"- 平均：{value:.2f}\n"
            report += f"- 最大：{analysis.get(f'{col}_max', 'N/A')}\n"
            report += f"- 最小：{analysis.get(f'{col}_min', 'N/A')}\n\n"

    return report

def main():
    print("📊 数据处理脚本启动...")
    data = load_csv("./data/input.csv")
    analysis = analyze_data(data)
    report = generate_report(data, analysis)

    with open("./分析报告.md", "w", encoding="utf-8") as f:
        f.write(report)
    print("✅ 分析完成！报告已保存到「分析报告.md」")

if __name__ == "__main__":
    main()
```
