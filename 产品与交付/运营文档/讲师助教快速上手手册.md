## 四、开场话术模板（三阶通用微调版）

### 讲师开场（5分钟）

> "大家好，欢迎来到未来AI实战训练营。今天不是来听课的，是来干活的。
>
> 我先问大家一个问题：你用过AI吗？用过的举手。——好，用过ChatGPT的？用过豆包的？用过但没做出过什么完整东西的？——没关系，**{今天/这X天}**结束的时候，你们每个人都会带着**{成果物数量}**个东西离开：**{列举成果物}**。
>
> 这不是理论课。**{今天/全程}**动手。你有电脑，打开就是。"

### 分赛道切换话术

**青少年版（替换括号内容）：**
- 为什么要学 → "10年后所有工作都会用AI，现在学就是提前起跑"
- 案例选择 → 学习场景（错题分析、作文辅助、科学实验设计）
- 激励方式 → "你的同学们都不会，你先学会了就是超人"

**成人版（替换括号内容）：**
- 为什么要学 → "你的同事已经在用AI写周报了，你还在手动写？"
- 案例选择 → 工作场景（邮件、周报、PPT、数据分析）
- 激励方式 → "今天学的东西明天上班就能用"

## 五、助教核心职责速查

### 开场前（-30分钟到0）

- [ ] 逐排检查学员工具登录状态
- [ ] 未登录的立刻协助（不要等开课）
- [ ] 确认Wi-Fi连通（打开任意一个AI工具测试）
- [ ] 备用热点放桌上，随时可切换
- [ ] 确认投屏/投影正常

### 每个模块期间

- [ ] **巡场走动**，不要坐着
- [ ] 看谁卡住了（屏幕上停了超过2分钟 → 上前帮助）
- [ ] **不替学员操作**，引导他们自己输入（"你试试这样描述……"）
- [ ] 记录学员常见卡点 → 课后反馈给讲师

### 休息期间

- [ ] 重点帮助进度慢的学员
- [ ] 检查工具是否正常（有些会掉线）
- [ ] 确认下个模块的工具已准备好

### 收尾阶段

- [ ] 协助学员导出/保存成果物
- [ ] 收集学员反馈（可以用微信群接龙）
- [ ] 引导学员加群/关注公众号

## 六、高频应急处理速查

| 情况 | 处理方式 | 时间限制 |
|------|----------|----------|
| **工具打不开/加载慢** | 先刷新→换浏览器→用备用热点→用备用账号 | 3分钟内解决 |
| **Wi-Fi断了** | 立刻切换备用4G热点（助教提前开好） | 1分钟内切换 |
| **学员跟不上进度** | 助教到身边一对一引导，不要等 | 即时处理 |
| **学员电脑没带/坏了** | 启用备用设备，或与邻座同学共用 | 5分钟内安排 |
| **工具到免费额度上限** | 切换备用账号，或换同类工具（豆包↔DeepSeek） | 2分钟内切换 |
| **讲师设备故障** | 助教设备顶上，讲师口头讲解+助教投屏操作 | 3分钟内切换 |
| **学员提前完成** | 给进阶挑战任务（优化作品/帮助邻座/探索新功能） | 即时 |

## 八、双赛道快速切换指南

### 青少年赛道特征

- **语言风格：** 活泼、鼓励、用"超人""酷""厉害"等词
- **案例方向：** 学习辅助（错题、作文、实验）、兴趣探索（游戏设计、动漫、音乐）
- **分组方式：** 按兴趣/学校混搭，4-5人一组
- **家长沟通：** 课前发家长须知、课中拍照发群、课后发成果物+学习报告
- **安全注意：** 不收集个人信息、不暴露真实姓名、AI对话内容需过滤

### 成人赛道特征

- **语言风格：** 专业、直接、用效率/成本/结果导向词汇
- **案例方向：** 工作效率（邮件、周报、PPT）、业务场景（营销、客服、数据分析）
- **分组方式：** 按行业/岗位混搭，4-5人一组
- **成果导向：** 强调"明天上班就能用"
- **时间敏感：** 成人更在意时间效率，减少闲聊，增加实操

### 切换口诀

> 青少年讲"未来你能做什么"，成人讲"现在你就能做什么"
> 青少年多鼓励"你太厉害了"，成人多认可"这个方法确实高效"
> 青少年用游戏化激励，成人用ROI激励

## 十一、常用话术速查

### 学员卡住时

- "没关系，AI就是这样，要多试几次。你换个说法试试？"
- "你先把你想做的用一句话写下来，然后直接发给AI"
- "你看看旁边同学怎么做的，互相学习一下"

### 学员提前完成时

- "很棒！你可以试试优化一下你的作品，比如加个配色/换个风格"
- "你可以帮帮旁边的同学，教别人也是学习"
- "你试试用另一个AI工具做同样的事，看看效果有什么不同"

### 升阶引导时（收尾阶段）

- "今天你学的是AI的基础用法。如果你想更深入——比如用AI建网站、做Agent——我们有二阶课程"
- "今天做出的东西可以发朋友圈，你的朋友一定会问你在哪学的"
- "你已经比90%的人先走了一步，接下来要不要继续？"

### 转化话术（自然版）

- "今天是体验课，完整版会带你从0到1做出真正能用的东西"
- "二阶不光教你用工具，还教你把AI变成你自己的产品"
- "今天报名有早鸟价，详情可以找我们的工作人员"
