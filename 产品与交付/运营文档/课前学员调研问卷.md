## 三、问卷使用 SOP

### 3.1 发送流程

| 步骤 | 时间 | 动作 | 负责人 |
|------|------|------|--------|
| 1 | 报名确认当天 | 发送问卷链接到学员微信 | 招生顾问 |
| 2 | 发送后第2天 | 检查回收情况，未填的一对一提醒 | 招生顾问 |
| 3 | 开营前5天 | 截止回收，未回收的电话回访 | 招生顾问 |
| 4 | 开营前3天 | 导出数据，交给讲师和助教 | 运营 |
| 5 | 开营前2天 | 根据问卷结果分组、准备个性化内容 | 讲师 |

### 3.2 数据分析模板

收到问卷后，整理以下关键数据交给讲师：

```

📊 职业分布（成人）：
- 企业员工：___人
- 管理层：___人
- 自由职业：___人
- 创业者：___人
- 其他：___人

📊 AI认知水平：
- 完全新手：___人（__%）
- 略有了解：___人（__%）
- 经常使用：___人（__%）
- 深度使用：___人（__%）

📊 最想学的内容TOP3：
1. ___（___人）
2. ___（___人）
3. ___（___人）

📊 设备情况：
- 有笔记本电脑：___人（__%）
- 只有手机/平板：___人（__%）
- 需要借用设备：___人（__%）

📊 特殊需求：
- [列出所有开放题中的特殊需求]
```

### 3.3 基于问卷结果的调整建议

| 发现 | 调整动作 |
|------|----------|
| 新手占比 > 70% | 增加基础操作演示时间，减少高级内容 |
| 设备不足 | 准备备用笔记本，提前测试投屏 |
| 某个工具需求集中 | 该工具教学时间增加20% |
| 青少年年龄段偏低 | 简化话术，增加趣味互动 |
| 成人企业客户多 | 增加企业场景案例 |
| 网络担忧多 | 确认场地WiFi带宽，准备4G热点备用 |
