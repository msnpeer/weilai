# 未来 AI 实战训练营 · 学员报名登记表

> **用途：** 招生启动后（8/4起）的标准化学员信息登记与管理工具
> **生成时间：** 2026-07-05 · 第47轮
> **适用范围：** 首期招生及后续期次，双赛道通用
> **使用方式：** 建议在腾讯文档/飞书多维表格中创建在线版本，多人实时协作

---

## 一、使用说明

### 1.1 本表解决什么问题

招生启动后，一线招生人员需要一个统一的地方记录每个学员的报名信息。之前已有《招生数据看板模板》追踪宏观数据，但缺少**逐人登记**的微观工具。本表补齐这个缺口。

### 1.2 与其他文档的关系

| 文档 | 作用 | 数据流向 |
|------|------|----------|
| 本表（学员报名登记表） | 逐人登记报名信息 | → 汇总到数据看板 |
| 招生数据看板模板 | 宏观数据追踪（人数/渠道/转化率） | ← 从本表汇总 |
| 学员分组管理模板 | 开营前按问卷结果分组 | ← 从本表获取已缴费学员名单 |
| 课前学员调研问卷 | 了解学员背景/设备情况 | → 补充本表的问卷回收状态 |
| 招生转化话术SOP | 咨询跟进话术 | 配合本表的跟进记录使用 |

### 1.3 填写规则

- **谁填：** 负责招生的一线人员（含城市合伙人）
- **何时填：** 每次有新咨询/报名时立即录入
- **更新频率：** 实时更新，每日核对
- **隐私要求：** 本表含学员个人信息，仅项目团队内部使用，不得外泄

## 七、操作SOP

### 7.1 新咨询录入流程

```
1. 收到咨询 → 立即在对应赛道表中新增一行
2. 填写：编号/姓名/手机/城市/渠道/咨询日期
3. 跟进记录栏填写首次跟进内容
4. 状态设为"咨询中"
5. 同步更新汇总看板的"咨询量"
```

### 7.2 报名缴费流程

```
1. 学员确认报名 → 更新状态为"已报名"
2. 填写报名阶段/报名日期
3. 发送缴费指引（⏳ 待确认收款账户信息）
4. 收到缴费 → 更新状态为"已缴费"
5. 填写缴费金额/方式/日期
6. 同步更新汇总看板的"已缴费"
7. 发送确认消息 + 课前问卷链接
```

### 7.3 分组前检查流程

```
1. 招生截止日前3天 → 导出所有"已缴费"学员
2. 核对问卷回收状态（配合《课前学员调研问卷》SOP）
3. 未回收问卷 → 电话回访（参考《学员分组管理模板》）
4. 确认设备情况 → 标记需借设备的学员
5. 将完整数据导入《学员分组管理模板》进行分组
```

### 7.4 数据安全要求

```
1. 本表仅限项目团队内部使用
2. 在线文档设置为"仅协作者可编辑"
3. 不得将学员手机/微信号分享给无关人员
4. 招生结束后，纸质版需销毁或归档
5. 青少年学员信息特别注意家长隐私保护
```

## 九、在线协作建议

### 9.1 推荐工具

| 工具 | 优势 | 适用场景 |
|------|------|---------|
| 腾讯文档-多维表格 | 免费/多人协作/手机可编辑 | 首选，一线招生人员易上手 |
| 飞书多维表格 | 功能更强/自动化/视图丰富 | 有飞书团队的合伙人 |
| Google Sheets | 国际化/公式强 | 有翻墙需求时 |
| Excel/WPS | 离线可用 | 备用方案 |

### 9.2 多视图建议

在多维表格中创建以下视图，方便不同角色查看：

| 视图名称 | 筛选条件 | 使用角色 | 用途 |
|---------|---------|---------|------|
| 全部学员 | 无筛选 | 运营负责人 | 全局查看 |
| 待跟进 | 状态=咨询中 | 招生人员 | 每日跟进清单 |
| 待缴费 | 状态=已报名 | 招生人员 | 催缴提醒 |
| 已缴费名单 | 状态=已缴费 | 运营/讲师 | 开营前核对 |
| 青少年赛道 | 赛道=青少年 | 青少年班负责人 | 单赛道查看 |
| 成人赛道 | 赛道=成人 | 成人班负责人 | 单赛道查看 |
| 渠道分析 | 按渠道分组 | 运营负责人 | 渠道效果分析 |

### 9.3 通知设置

- 新增报名 → 通知运营负责人
- 缴费状态变更 → 通知财务+运营
- 超24h未缴费 → 自动提醒招生人员跟进
- 招生人数达到目标 → 全员通知
