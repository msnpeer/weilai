# 赛事体系设计方案

> 未来 AI 实战训练营 · 赛事与竞技体系
> 版本：草稿 v1.0 · 2026-07-02

---

## 一、赛事体系定位

### 1.1 为什么需要赛事体系

| 价值维度 | 说明 |
|----------|------|
| **招生钩子** | "参加AI大赛获奖"比"上了AI课"更有传播力，尤其青少年赛道——家长转发率极高 |
| **品牌势能** | 赛事自带"权威感"和"规模感"，是品牌升级的关键一步 |
| **学员黏性** | 赛事创造"再来一次"的理由——获奖者自然想参加更高阶赛事，未获奖者想再战 |
| **城市合伙人赋能** | 赛事给合伙人一个"本地活动"抓手，比单纯招生更有话题性 |
| **商业延伸** | 赛事可衍生出赞助、冠名、评委、媒体报道、学校合作等多种商业模式 |

### 1.2 赛事体系原则

1. **实战导向**：比赛内容必须是真实AI应用产出，不是理论答题
2. **三阶联动**：赛事与L1/L2/L3课程体系挂钩，形成"学→赛→学"循环
3. **双轨并行**：青少年赛道和成人赛道分开设置赛事主题和评审标准
4. **低门槛高上限**：一阶学员也能参赛，三阶学员有专项挑战
5. **城市可落地**：赛事不依赖总部大资源，城市合伙人可独立组织

## 二、赛事类型总览

### 2.1 三级赛事体系

| 级别 | 名称 | 频率 | 范围 | 定位 |
|------|------|------|------|------|
| **L1 赛事** | AI 创作挑战赛 | 每期训练营结营日 | 单营内部 | 一阶结营作品评选，激发成就感 |
| **L2 赛事** | AI 造物大赛 | 每季度/每2个月 | 城市级 | 二阶学员为主，开放外部报名，扩大影响力 |
| **L3 赛事** | AI 系统创新赛 | 每半年/年度 | 全国级 | 三阶学员精英赛，年度品牌盛事 |

### 2.2 赛事阶梯逻辑

```
L1 创作挑战赛（每期结营）
  ├→ 获奖者推荐参加 L2 造物大赛
  └→ 参赛体验驱动升阶报名

L2 造物大赛（季度城市赛）
  ├→ 获奖者推荐参加 L3 系统创新赛
  ├→ 外部参赛者转化为二阶/三阶学员
  └→ 城市合伙人可独立组织

L3 系统创新赛（半年/年度全国赛）
  ├→ 年度品牌盛事
  ├→ 获奖者进入"未来AI精英社群"
  └→ 优秀作品可商业化孵化
```

## 三、L1 赛事：AI 创作挑战赛

### 3.1 赛事定位

| 维度 | 内容 |
|------|------|
| 时间 | 每期一阶训练营最后1小时（结营环节） |
| 参赛者 | 当期一阶全体学员（强制参与，非可选） |
| 形式 | 现场限时作品展示 + 小组互评 + 讲师评选 |
| 目的 | 让学员带着"获奖体验"离开，提升满意度和口碑传播 |

### 3.2 赛事规则

#### 青少年赛道

| 项目 | 说明 |
|------|------|
| 比赛内容 | 用当天所学AI工具，完成一个完整作品（海报+短视频+提示词） |
| 限时 | 30分钟（在原有课程时间中预留） |
| 评审标准 | 创意性（30%）+ 完整度（30%）+ AI工具运用（20%）+ 展示表达（20%） |
| 评审方式 | 小组互评（50%）+ 讲师评选（50%） |
| 奖项设置 | 最佳创意奖 / 最佳视觉奖 / 最佳表达奖 / 全能之星（各1名） |
| 奖品 | 电子获奖证书 + 品牌周边（贴纸/徽章） + 二阶课程优惠券 |

#### 成人赛道

| 项目 | 说明 |
|------|------|
| 比赛内容 | 用当天所学AI工具，完成一个业务场景作品（宣传图+短视频+提示词模板） |
| 限时 | 30分钟 |
| 评审标准 | 实用性（30%）+ 完整度（30%）+ AI工具运用（20%）+ 商业价值（20%） |
| 评审方式 | 学员互评（50%）+ 讲师评选（50%） |
| 奖项设置 | 最佳实用奖 / 最佳创意奖 / 最佳效率奖 / AI先锋（各1名） |
| 奖品 | 电子获奖证书 + 二阶课程优惠券 + 1次免费AI咨询（30分钟） |

### 3.3 赛事流程

```
09:00-09:10  赛事说明 + 规则讲解
09:10-09:40  学员限时创作（30分钟）
09:40-10:00  小组展示 + 互评投票
10:00-10:10  讲师点评 + 颁奖
10:10-10:20  获奖感言 + 合影
```

### 3.4 赛事产出物

- 获奖学员作品集（用于后续招生宣传）
- 赛事照片/视频素材（用于朋友圈/公众号）
- 获奖证书电子版（学员朋友圈分享）
- 赛事数据（参赛率、获奖率、好评率）

## 四、L2 赛事：AI 造物大赛

### 4.1 赛事定位

| 维度 | 内容 |
|------|------|
| 时间 | 每季度或每2个月一次，具体由城市合伙人确定 |
| 参赛者 | 二阶学员优先，一阶学员可参加初级赛道，外部人士可报名体验赛道 |
| 形式 | 线上提交作品 + 线下路演展示（可选） |
| 目的 | 扩大品牌影响力、转化外部学员、展示二阶教学成果 |

### 4.2 赛道设置

#### 赛道一：学员赛道（已完成二阶课程）

| 项目 | 说明 |
|------|------|
| 比赛主题 | 每次指定主题方向（如"教育AI产品"、"商业展示网站"、"个人品牌Agent"） |
| 作品要求 | 用二阶所学工具（Agent+网站+自动化）完成一个完整可访问的数字产品 |
| 提交物 | 可访问网站链接 + Agent演示录屏 + 作品说明文档 |
| 评审标准 | 产品完整度（25%）+ 创意（25%）+ 技术实现（25%）+ 商业/实用价值（25%） |
| 评审方式 | 总部讲师团（40%）+ 城市合伙人（30%）+ 大众投票（30%） |

#### 赛道二：开放赛道（未参加过二阶课程）

| 项目 | 说明 |
|------|------|
| 比赛主题 | 与学员赛道相同，降低标准 |
| 作品要求 | 用任意AI工具完成一个数字作品（网站/海报/短视频/文案均可） |
| 提交物 | 作品链接或文件 + 使用工具说明 |
| 评审标准 | 创意（40%）+ 完整度（30%）+ AI工具运用（30%） |
| 评审方式 | 城市合伙人评审（60%）+ 大众投票（40%） |

### 4.3 奖项设置

| 奖项 | 名额 | 奖品 |
|------|------|------|
| 金奖（学员赛道） | 1名 | 获奖证书 + 三阶课程全额奖学金 + 品牌大使资格 |
| 银奖（学员赛道） | 2名 | 获奖证书 + 三阶课程50%优惠 + 品牌周边礼盒 |
| 铜奖（学员赛道） | 3名 | 获奖证书 + 三阶课程30%优惠 + 品牌周边 |
| 最佳创意奖 | 1名 | 获奖证书 + 品牌周边 |
| 最佳人气奖 | 1名 | 获奖证书 + 品牌周边 |
| 开放赛道优秀奖 | 3名 | 获奖证书 + 二阶课程体验券 |

### 4.4 赛事运营流程

#### 赛前（4-6周）

| 时间 | 动作 | 负责方 |
|------|------|--------|
| 赛前6周 | 确定赛事主题、时间、评审团 | 总部 |
| 赛前5周 | 发布赛事海报和报名通道 | 总部+城市合伙人 |
| 赛前4周 | 开放报名，开始赛事宣传 | 城市合伙人 |
| 赛前2周 | 截止报名，发送参赛指南 | 总部 |
| 赛前1周 | 参赛者提交作品初稿，导师1对1反馈（学员赛道） | 总部讲师 |

#### 赛中（1-2天）

| 时间 | 动作 |
|------|------|
| Day1 上午 | 开幕式 + 赛事说明 + 分组 |
| Day1 下午 | 作品优化冲刺（现场支持） |
| Day2 上午 | 路演展示（每个作品3-5分钟） |
| Day2 下午 | 评审打分 + 颁奖典礼 + 合影 |

#### 赛后（1-2周）

| 时间 | 动作 | 负责方 |
|------|------|--------|
| 赛后1周 | 获奖作品公众号/朋友圈宣传 | 总部 |
| 赛后1周 | 获奖者1对1沟通三阶转化 | 城市合伙人 |
| 赛后2周 | 赛事复盘报告 | 总部 |

### 4.5 赛事产出物

- 获奖作品展示页面（官网子页面）
- 赛事宣传视频（3-5分钟精华版）
- 参赛者数据统计（报名数/完赛率/转化率）
- 赛事复盘报告

## 五、L3 赛事：AI 系统创新赛

### 5.1 赛事定位

| 维度 | 内容 |
|------|------|
| 时间 | 每半年或每年一次（建议每年暑假 + 春节前） |
| 参赛者 | 三阶学员为主，优秀二阶学员可申请 |
| 形式 | 线上方案提交 + 线下终审路演（可设在试点城市轮办） |
| 目的 | 年度品牌盛事、孵化优秀AI项目、打造"未来AI"精英社群 |

### 5.2 比赛形式

#### 阶段一：方案征集（线上，4周）

| 项目 | 说明 |
|------|------|
| 比赛主题 | 每年指定大方向（如"AI改变教育"、"AI重塑本地生活"、"AI赋能中小企业"） |
| 提交物 | AI系统方案书（含需求分析+工具选型+系统架构+预期效果）+ 原型Demo |
| 评审标准 | 方案完整性（20%）+ 创新性（25%）+ 可行性（25%）+ 商业/社会价值（30%） |

#### 阶段二：初审筛选（1周）

- 总部讲师团评审，选出 Top 10 进入终审
- 未入选者获得"优秀方案奖"电子证书

#### 阶段三：终审路演（线下，1天）

| 时间 | 内容 |
|------|------|
| 上午 | Top 10 路演展示（每人10分钟展示 + 5分钟答辩） |
| 下午 | 评审闭门会 + 颁奖典礼 + 精英社群成立仪式 |

### 5.3 评审团构成

| 角色 | 人数 | 来源 |
|------|------|------|
| 总部讲师代表 | 2人 | 未来AI训练营总部 |
| 行业专家 | 1-2人 | AI/教育/投资领域（外邀） |
| 城市合伙人代表 | 1人 | 轮值 |
| 往届获奖学员 | 1人 | L2赛事金奖获得者 |

### 5.4 奖项设置

| 奖项 | 名额 | 奖品 |
|------|------|------|
| 冠军 | 1名 | 获奖证书 + 现金奖金（⏳待定金额）+ 项目孵化支持 + 年度品牌大使 |
| 亚军 | 1名 | 获奖证书 + 现金奖金（⏳待定金额）+ 项目孵化支持 |
| 季军 | 1名 | 获奖证书 + 现金奖金（⏳待定金额） |
| 最佳创新奖 | 1名 | 获奖证书 + 品牌周边礼盒 |
| 最佳商业价值奖 | 1名 | 获奖证书 + 品牌周边礼盒 |
| 最受大众欢迎奖 | 1名 | 获奖证书 + 品牌周边礼盒 |

### 5.5 项目孵化支持（冠军/亚军专属）

- 总部提供1个月免费技术指导（每月2次1小时咨询）
- 帮助对接潜在合作方或客户
- 优秀作品可纳入"未来AI"案例库展示
- 提供品牌背书支持（可用于获奖者自己的商业宣传）

## 六、赛事与课程体系联动

### 6.1 "学→赛→学"循环设计

```
学员参加一阶 → 结营参加L1挑战赛 → 获奖/体验好 → 报名二阶
学员完成二阶 → 参加L2造物大赛 → 获奖/体验好 → 报名三阶
学员完成三阶 → 参加L3系统创新赛 → 获奖 → 进入精英社群/成为助教
```

### 6.2 赛事作为招生转化工具

| 赛事 | 招生转化作用 |
|------|-------------|
| L1 挑战赛 | 一阶结营的"高光时刻"，学员朋友圈分享 → 朋友看到 → 新生报名 |
| L2 造物大赛 | 开放赛道吸引外部人士 → 体验后转化为正式学员 |
| L3 系统创新赛 | 品牌盛事 → 媒体报道 → 品牌知名度提升 → 间接招生 |

### 6.3 赛事与认证体系联动

| 联动点 | 说明 |
|--------|------|
| 获奖加分 | L1/L2赛事获奖者在对应阶认证中获得"优秀"评定加分 |
| 徽章解锁 | L2赛事金奖获得者可解锁"路演之星"专项徽章 |
| L3赛事参与者 | 获得"AI创新者"专属徽章（不在常规徽章体系中，赛事专属） |

## 七、赛事与城市合伙人协同

### 7.1 城市合伙人在赛事中的角色

| 赛事级别 | 合伙人职责 |
|----------|-----------|
| L1 挑战赛 | 协助当期训练营结营组织、拍照、朋友圈宣传 |
| L2 造物大赛 | 本地赛事组织方：场地、宣传、报名、初审、活动执行 |
| L3 系统创新赛 | 推荐本地优秀学员参赛、协助宣传 |

### 7.2 城市合伙人可独立组织的赛事

**L2 造物大赛** 可由城市合伙人独立组织，总部提供：

- 统一赛事海报模板
- 统一评审标准和评分表
- 统一奖项证书模板
- 总部讲师团线上评审支持
- 赛事宣传文案模板

城市合伙人需自行准备：

- 场地（可与训练营场地复用）
- 本地宣传和报名渠道
- 活动现场执行
- 本地评审（1-2名）
- 奖品采购（除证书外）

### 7.3 赛事收益分配

| 收益项 | 总部 | 城市合伙人 |
|--------|------|-----------|
| 赛事报名费（开放赛道） | 30% | 70% |
| 赞助收入 | 50% | 50% |
| 赛事转化学员报名费 | 按正常城市合作协议分润 | 按正常城市合作协议分润 |

> ⏳ 赛事报名费定价和赞助分成比例需老板确认

## 十、赛事风险与应对

### 10.1 风险识别

| 风险 | 可能性 | 影响 | 应对措施 |
|------|--------|------|---------|
| 参赛人数不足 | 中 | 降低赛事氛围 | L1强制参与；L2设开放赛道吸引外部；提前宣传 |
| 作品质量参差不齐 | 高 | 评审困难 | 设"最低合格线"，低于线的作品不参与评奖 |
| 抄袭/代做 | 低 | 品牌风险 | 要求提交工具使用录屏；现场答辩验证真实性 |
| 评审争议 | 中 | 口碑风险 | 多维评审（讲师+合伙人+大众）；设申诉机制 |
| 赞助拉不到 | 高 | 收入低于预期 | 首期不依赖赞助收入；赛事成本控制在低成本范围 |
| 城市合伙人组织能力不足 | 中 | 赛事体验差 | 总部提供全套物料+流程SOP；首期总部派人现场支持 |
