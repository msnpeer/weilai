## 二、话术一：首次接触话术

> **场景：** 朋友圈看到内容后私信 / 公众号留言 / 朋友推荐后加微信 / 线下活动扫码

### 2.1 通用开场（30秒内建立专业感）

```
你好！我是未来 AI 实战训练营的 [名字]，感谢你的关注 😊

我们是全程实操的 AI 课程——不是坐着听讲，而是人手一台 MacBook，
现场做出真实作品：海报、短视频、网站，甚至自动化工具。

你是自己感兴趣，还是想给孩子/团队了解一下？
```

### 2.2 自我需求判断分流

| 对方回答 | 判断 | 转入话术 |
|----------|------|----------|
| "我自己想学" | 成人赛道 | → 话术二·成人版 |
| "想给孩子报" | 青少年赛道 | → 话术二·青少年版 |
| "公司想组织员工学" | 企业客户 | → 话术二·企业版 |
| "先了解一下" | 中性/观望 | → 话术二·通用版 |
| "帮我看看哪个合适" | 有意向 | → 话术二·推荐版 |

## 三、话术二：需求摸底话术

> **场景：** 对方产生兴趣后，需要快速判断其需求、匹配合适阶段、建立信任

### 3.1 通用摸底三问

```
问1：你之前有用过 ChatGPT / 豆包 / DeepSeek 这类 AI 工具吗？
（判断：零基础 → 一阶 / 有基础 → 可推二三阶）

问2：你最想用 AI 解决什么问题？是做内容、做产品，还是提升工作效率？
（判断：兴趣方向 → 匹配对应阶段卖点）

问3：你大概什么时候方便？我们首期 10/1 国庆开营，每期名额有限。
（判断：时间意愿 → 推进报名节奏）
```

### 3.2 成人赛道 · 分阶段推荐话术

#### 一阶推荐（零基础 / 初次接触 AI）

```
那我推荐你从一阶·AI 破界营开始，成人3天（青少年5天），从零基础到亲手做出：

✅ 1 个 AI 原创海报
✅ 1 条 30 秒 AI 短视频
✅ 1 套你自己的提示词模板

全程老师带着操作，不需要任何基础。现场提供 MacBook，你只带好奇心就行。

价格是 ¥3,980，首期有早鸟优惠。你想了解具体哪一期？
```

#### 二阶推荐（有基础 / 想做产品）

```
你有基础的话，二阶·AI 造物营更适合你。青少年7天（成人3天），你会亲手做出：

✅ 1 个真正能打开、能访问的网站
✅ 1 套 AI 辅助建站的 SOP（以后自己能重复用）
✅ 1 个能跑通的专属 Agent

不是学概念，是做出一个真正的产品。价格是 ¥9,800。

你之前有做过网站或者用过 Agent 类工具吗？
```

#### 三阶推荐（深度需求 / 想搭系统）

```
如果你是想把 AI 真正用进工作流，三阶·AI 掌舵营是最高阶的。3天时间：

✅ 1 套结构化知识库（你的专属 AI 大脑）
✅ 1 个自动化脚本或轻量工具
✅ 1 套 AI 系统落地方案（可以直接带回公司用）

价格是 ¥19,800，适合企业中高管、创业者、想深度转型的团队。

你目前工作中最想用 AI 自动化的是哪个环节？
```

### 3.3 青少年赛道 · 推荐话术

```
了解！给孩子的 AI 教育确实越来越重要了。

我们的青少年课程和成人是同一个体系，但案例和任务会更贴近孩子：
```

## 四、话术三：异议处理话术

> **场景：** 对方有兴趣但有顾虑，需要逐一打消

### 4.1 高频异议 TOP 10

#### 异议 1："太贵了"

```
理解你的顾虑。我给你算一笔账：

一阶 ¥3,980，成人3天（青少年5天），你带走的不只是体验，而是一套可以直接用的 AI 工作流。
一个设计师做一张海报收费 500-2000，你学会了以后自己做，成本趋近于零。

而且我们的定价已经包含了：MacBook 使用、全套工具、讲师指导、课后社群。
外面学个 PPT 课程都要好几千，这个性价比其实很高。

你是对哪一阶的价格有顾虑？我可以帮你看看有没有更合适的方案。
```

#### 异议 2："我怕学不会"

```
这是最常见的顾虑，也是我们最重视的问题。

我们的设计原则就是"零基础可学"：
- 每一步老师先演示，你跟着做
- 助教全程巡场，卡住了随时举手
- 不是讲完理论让你自己悟，是一步步带你做出来

上一期学员 90% 都是零基础，最后每个人都带走了成品。

你最担心的是哪个部分？我帮你看看具体是怎么安排的。
```

#### 异议 3："我自己在网上学不行吗"

```
当然可以，网上资源很多。但你有没有这个体验：

看了一堆教程，收藏了一堆文章，但真正动手做的时候还是不知道从哪开始？

我们的价值不只是教你工具，而是：
1. **结构化路径** — 从哪开始、按什么顺序、做到什么程度，帮你设计好了
2. **现场实操** — 1天/2天/3天集中突破，不给自己拖延的机会
3. **真实产出** — 你不是"学了一遍"，而是"做出来了"
4. **社群陪伴** — 课后有问题还能问，不是学完就断

你之前自学 AI 大概学了多久？感觉效果怎么样？
```

#### 异议 4："国庆想休息/有安排"

```
完全理解，国庆确实想放松。不过换个角度想：

1天时间，别人在景区排队的时候，你已经掌握了一套能用一整年的 AI 工作流。
而且我们的时间安排不会占用整个假期——青少年一阶5天、二阶7天、三阶7天，成人均为3天。

你可以选择 10/1-10/3 中的任意一天或连续几天，灵活安排。

你国庆大概哪几天有空？我帮你看看排期。
```

#### 异议 5："AI 发展太快，学了会不会很快过时"

```
这个问题问得很好，说明你是认真思考过的。

确实，AI 工具在快速迭代。但我们的课程核心不是教你"某个工具怎么点"，
而是教你"怎么用 AI 解决问题"的方法论和思维方式。

举个例子：
- 我们教你"怎么写好提示词"，不管工具怎么变，这个能力永远有用
- 我们教你"怎么搭建系统"，底层逻辑是一样的
- 我们教你"怎么把 AI 融入工作流"，这个思路不会过时

而且课后我们有社群，工具更新了大家会一起交流新玩法。

你最想用 AI 解决的场景是什么？我帮你看看学了以后能直接用上。
```

#### 异议 6："我要跟家人/朋友商量一下"

```
当然，这么重要的决定确实值得商量。

我先把详细资料发给你，你可以转给家人/朋友看看。
有任何问题随时找我，我在线解答。

另外提醒一下，我们首期名额有限，每期小班制，
如果你确定要参加的话建议尽早预留名额。

你方便把我的名片推给家人吗？有什么问题可以直接问我。
```

#### 异议 7："我年纪大了/太小了，能学吗"

```
青少年赛道：
我们有专门针对 10-17 岁青少年的课程设计，案例和任务都会适配年龄段。
助教会全程陪伴，确保每个孩子都能跟上。

成人赛道：
上一期有 50 多岁的学员，最后做得比年轻人还快。
AI 的好处就是门槛低——你不需要会编程，会打字就行。
而且成年人的理解力和目标感更强，学起来反而更快。

你（/孩子）今年多大？我帮你看看最适合从哪一阶开始。
```

#### 异议 8："我在外地，不方便来"

```
我们目前在山东、重庆、珠海、海南都有试点城市。
你看看哪个城市离你最近？

另外，如果你所在的城市有我们的城市合伙人，
他们可能会组织本地的训练营，更方便参加。

你目前在哪个城市？我帮你查一下最近的安排。
```

#### 异议 9："先观望一下，看看别人学完怎么样"

```
观望也是一种策略。但有两个提醒：

1. 首期的价格是最优惠的，后续会恢复原价
2. 首期学员会有专属社群和后续福利，错过就没有了

如果你想先看看效果，可以关注我们的公众号/朋友圈，
首期结束后我们会分享学员成果和反馈。

不过如果真的感兴趣，我建议你至少先预留一个名额——
确定不了的话，开营前 7 天可以无条件退款。

这样你既锁定了名额和优惠，又没有风险。
```

#### 异议 10："我就是随便看看"

```
没关系，随便看看也欢迎 😊

我先发一份详细资料给你，你有空的时候看看。
有任何问题随时找我，不着急。

顺便说一下，我们 10/1 国庆首期正在招生中，
如果你身边有朋友感兴趣也可以推荐给我，我们有转介绍福利。
```

## 五、话术四：促成报名话术

```
1. 姓名：
2. 联系电话：
3. 选择阶段：一阶 / 二阶 / 三阶

5. 所在城市：

登记完我发你缴费链接，缴费成功后名额就锁定了。
```

```
提醒一下，我们首期每期名额有限，小班制教学。
目前 [城市] 的 [阶段] 已经报了 XX 人了，还剩 X 个名额。

你确定要参加的话，建议今天登记锁定名额。
开营前 7 天可以无条件退款，完全没有风险。
```

### 5.3 优惠促成法

```
首期报名有早鸟优惠：
- 一阶：原价 ¥X，早鸟价 ¥X（优惠 ¥X）
- 二阶：原价 ¥X，早鸟价 ¥X（优惠 ¥X）
- 三阶：原价 ¥X，早鸟价 ¥X（优惠 ¥X）
```

### 5.5 退路促成法（降低决策压力）


## 十、话术使用指南

### 10.1 使用原则

1. **先听后说** — 先了解对方需求，再推荐对应阶段
2. **真诚自然** — 不要念稿，用自己的话说，但核心信息不能漏
3. **制造紧迫** — 名额有限、早鸟优惠、首期福利，但不要过度施压
4. **留好退路** — 每次促成都要给对方"安全网"（无条件退款）
5. **及时跟进** — 24/48/72小时跟进节奏，不要让意向客户流失

### 10.2 禁忌

- ❌ 不要贬低竞品
- ❌ 不要过度承诺（"保证学会""保证就业"）
- ❌ 不要给不确定的优惠（超出权限的折扣需请示）
- ❌ 不要在非工作时间频繁打扰（微信消息 22:00-8:00 不发）
- ❌ 不要泄露其他学员的个人信息

### 10.3 常用表情/语气建议

- 适度使用 emoji（😊🎉✅）增加亲和力
- 感叹号不要超过1个
- 回复速度：工作时间 5 分钟内，非工作时间 1 小时内
- 长消息分段发，不要一屏塞满

### 10.4 各阶段转化率参考

| 转化节点 | 目标转化率 | 预警线 |

| 意向→报名 | ≥60% | <40% |

## 十一、与招生数据看板联动

每日填写《招生数据看板模板》时，需同步记录：

1. **新增咨询来源** — 记录渠道（朋友圈/公众号/合伙人/转介绍）
