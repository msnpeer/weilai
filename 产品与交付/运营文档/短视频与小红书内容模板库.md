## 使用说明

| 项目 | 说明 |
|------|------|
| 适用渠道 | 小红书（图文笔记）、抖音/视频号（短视频） |
| 适用人群 | 招生运营、城市合伙人、老板本人 |
| 内容分类 | 引流型（拉新）/ 种草型（转化）/ 见证型（信任） |
| 双赛道 | 青少年赛道（家长视角）/ 成人赛道（职场人视角） |
| 配合物料 | 海报HTML（poster-*.html）、落地页（landing-page.html）、招生文案成品 |

**核心原则（来自XHS爆款方法论）：**
- ❌ 不要广告感设计（渐变、科技感、精美排版）
- ❌ 不要堆砌所有卖点
- ✅ 要素人感（纯色背景 + 大emoji + 简单文字）
- ✅ 要口语化（"有没有"、"宝藏"、"好看哭了"）
- ✅ 要真实素材（课程现场截图、学员作品、实际效果）

## 四、内容制作指南

### 小红书笔记制作流程

1. **选题：** 从模板库中选择当期要发的笔记
2. **封面：** 按模板设计封面（纯色背景 + 大emoji + 大字）
3. **正文：** 按模板填写正文，替换具体信息
4. **标签：** 添加5-8个相关标签
5. **发布时间：** 建议 早8:00 / 午12:30 / 晚21:00
6. **互动：** 发布后1小时内回复所有评论

### 抖音短视频制作流程

1. **选脚本：** 从脚本库中选择当期要拍的视频
2. **拍摄：** 按脚本拍摄（面对镜头/屏幕录制）
3. **剪辑：** 添加字幕、BGM、转场
4. **发布：** 添加话题标签 + 定位
5. **互动：** 发布后1小时内回复所有评论
